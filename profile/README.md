# Núcleo de Inteligência de Dados – RioSaúde  
Repositório oficial do Núcleo de Inteligência de Dados da RioSaúde.

Somos responsáveis por desenvolver soluções analíticas, pipelines de dados, APIs, automações e dashboards estratégicos que apoiam a tomada de decisão em toda a rede de saúde do município do Rio de Janeiro.

---

## 🚀 O que desenvolvemos
- Modelos analíticos e estatísticos  
- Pipelines de dados (Python, SQL, dbt, ETL/ELT)  
- Dashboards estratégicos (Power BI)  
- Processos automatizados para integrações de sistemas  
- Monitoramento e qualidade de dados  
- APIs internas e externas  
- Ambientes de Data Warehouse e Data Lake  

---

## 🛠️ Tecnologias que utilizamos
- **Python** (Pandas, NumPy, Prophet, Requests, APIs)  
- **SQL / PostgreSQL**
- **dbt Core**
- **Docker & Docker Compose**
- **Git & GitHub**
- **Google Cloud / AWS (S3, Athena)**
- **Power BI**
- **Pipelines automatizados**
